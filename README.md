# slide-template

> Master keynote template for slides

## Install

1. Install Node.js (10.0.0 or later)
2. Clone this repository and `cd`
3. Install dependencies with `npm install`
4. Run `npm start`

## Features

- Uses a custom `tech-0` theme that compiles
- Settings to write external markdown instead of writing your slides inline

## Commands

- Hit `s` on your keyboard for speaker view.
- Hit `f` on your keyboard for fullscreen mode.
- Hit `esc` on your keyboard for overview view.
